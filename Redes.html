Estruturas Fundamentais

O que são Estruturas de Controle?

São blocos de código que determinam o fluxo de execução de um programa, permitindo decisões e repetições. Exemplos incluem:

  • If/Else: Escolhas baseadas em condições. 
    if (idade >= 18) {
    console.log("Você é maior de idade.");
} else {
    console.log("Você é menor de idade.");
}
  • Switch/Case: Alternativas para múltiplas condições. 
    switch(dia) {
    case 'Segunda':
        console.log("Início da semana!");
        break;
    default:
        console.log("Outro dia.");
}

Laços de Repetição

Por que usar Laços de Repetição?

Para automatizar tarefas repetitivas, como processar listas ou realizar cálculos em massa.

  • Enquanto-Faça: Executa enquanto a condição for verdadeira. 
    let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
  • Para-Faça: Define um número fixo de repetições. 
    for (let i = 0; i < 5; i++) {
    console.log(i);
}
  • Faça-Enquanto: Garante execução ao menos uma vez. 
    let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

Variáveis e Tipos de Dados

O que são Variáveis?

Espaços na memória usados para armazenar valores que podem mudar durante a execução do programa.

  • Inteiro: Números sem casas decimais (ex.: 1, 2, 100). 
    let idade = 25;
  • Real: Números com casas decimais (ex.: 3.14, 0.99). 
    let preco = 19.99;
  • Texto: Cadeias de caracteres (ex.: "Olá, Mundo!"). 
    let mensagem = "Bem-vindo!";

Funções

O que são Funções?

Blocos de código reutilizáveis que realizam uma tarefa específica. Podem receber parâmetros e retornar valores.

  • Função Simples: 
    function saudacao(nome) {
    return `Olá, ${nome}!`;
}
console.log(saudacao("Maria"));
  • Função Anônima: 
    const somar = function(a, b) {
    return a + b;
};
console.log(somar(5, 3));
  • Arrow Function: 
    const multiplicar = (a, b) => a * b;
console.log(multiplicar(4, 2));
